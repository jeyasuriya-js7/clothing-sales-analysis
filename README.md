# Clothing Sales Analysis

## Project Overview

This project analyzes a clothing sales dataset to uncover business
insights related to revenue, profit, customer behavior, product
performance, and sales trends. The project combines Python for data
analysis, SQL for business queries, and Power BI for dashboard creation.

## Dataset Information

The dataset contains sales transactions with information such as: -
Product details and categories - Customer information - Salesperson
performance - Sales locations - Sales channels - Revenue and profit
metrics - Order status and transaction dates

## Tools and Technologies

-   Python
-   Pandas
-   Matplotlib
-   Jupyter Notebook
-   SQL
-   Power BI

## Data Cleaning and Preparation

-   Checked dataset structure and summary statistics.

-   Identified missing values and duplicate records.

-   Converted `saleDate` into datetime format.

-   Created additional features:

    -   Year
    -   Month
    -   Month Number

-   Calculated profit using:

        Profit = Total Amount - Total Cost

-   Exported the cleaned dataset for further analysis.

## Exploratory Data Analysis (EDA)

The following analyses were performed: - Total sales and total profit
calculation - Profit percentage analysis - Product category
performance - Revenue by location - Top-performing salespersons -
Top-selling products - Sales trend analysis over time - Order status
analysis

## Visualizations Created

-   Top sales by location
-   Product category sales distribution (Pie Chart)
-   Top 5 salespersons by revenue
-   Top 10 products by revenue
-   Sales trends over time

## SQL Analysis

Business questions solved using SQL include: - Total revenue and total
profit - Most profitable products - Best product categories by
location - Categories exceeding sales thresholds - Top product
categories by profit - High-value customers - Locations with the highest
number of sales - Top customers by spending - Product ranking within
categories - Location ranking by revenue

## Power BI Dashboard

An interactive Power BI dashboard was created to visualize: - Revenue
and profit KPIs - Category-wise sales performance - Location analysis -
Salesperson performance - Product insights - Sales trends and patterns
